# -Smart-Energy-Meter-Dashboard
A sim# Smart Energy Meter Dashboard Simulation
A clean and lightweight Python application that models a digital smart electricity meter to track household power metrics and monitor safety thresholds.

---

## 🚀 Project Overview
This project simulates a modern digital electricity meter. The program generates real-time household electrical logs, handles core system computations, and tracks safety flags based on active electrical thresholds.

### Key Features
* **Automated Telemetry Stream:** Simulates continuous tracking cycles for standard household lines.
* **Electrical Math Tracking:** Computes real-time active wattage consumption ($Power = Voltage \times Current$).
* **Threshold Alert Logic:** Automatically triggers safety alert flags when electrical load thresholds are exceeded.

---

## 🛠️ Technology Stack
* **Programming Language:** Python 3.10+
* **Execution Interface:** Local Command Line Terminal
